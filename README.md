# api-webserver README

This extension make it possible to take control of Visual Studio Code from a Web Page based client.


## Features

POST request with data contents in json format:
```
{
    "files":[
        {
            "name":"Main.c",
            "contents":""
        },
        {
            "name":"Main.h",
            "contents":""
        }
    ],
    "command":""
}
```
